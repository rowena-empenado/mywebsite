ABOUT ME

My name is Ma. Rowena Empenado . I'm a Full Stack Web Developer based in Las Piñas, Philippines.

Prior to shifting into a Web Development career, I worked as an Analyst for several years in the financial services industry and a few years as a Software Tester in an IT consulting firm.

My passion for coding began when I started learning VBA back in 2021. Since then, I have created several Excel macro tools to automate BAU and admin tasks. I also started learning HTML and CSS while working on a project that streamlines our team's onboarding processes. From then on, I figured out that I love creating things that simplify people's lives.

Come 2022, I decided to pursue Web Development by enrolling to Zuitt's Full Stack Web Development bootcamp and there I gained the skills and experiences I need to become a well-equipped Web Developer.

E-commerce App

Capstone 3 project

This is an E-commerce MERN app (named as ShopPAW) built in 1-2 weeks.

Features:
- User registration page with proper form validation
- User authentication using jwt (JSON web token)
- Product catalog page and specific product page that varies depending on user type (regular user vs. admin)
- Cart view page that shows all the items the user has added to their cart, their quantities, subtotal, and the total price for all items ; user can also change product quantities, remove products from cart, and checkout the items
- Order page that shows all the user's previously placed orders
- Admin/Dashboard that can retrieve list of all products (available or unavailable), create new product, update product information, deactivate/reactivate product, and retrieve a list of all orders made by all users
- App is also responsive across mobile/tablet/desktop screen sizes
